# 1.5 常见问题

## 常见问题

1. **电机的canfd通讯的波特率和采样率配置的多少**

    答：仲裁段1m，数据段5m 。采样率是75%

2. **CAN用的什么信号**

    答：can是使用分差信号的，can_h是高电平信息线，can_l是低电平信号线。

3. **单独控制可以，同时控制多个就不行，**

    答：大概率是电源电流限制太小，多个电机一起启动瞬间电流太大，导致电机报错了

4. **电机有扭矩传感器吗？**

答：这个没有，这个是根据电机本身的电流环做的力控

1. **协议支持can2.0A还是can2.0B**

答：can2.0B

1. **终端电阻怎么测**

    答：断电测电阻，在一条CAN总线上电阻在60左右为正常，程序运行时测量结果没有意义。

2. **什么情况下需要配120Ω的电阻**

    答：尽量在CAN总线末端都接上120Ω的终端电机，这样可以减少CAN总线的干扰。

3. **6个电机串联的机械臂，需要用哪种电源。**

    答：最好用直流稳压电源，电压使用范围可以查看产品手册，电流尽量设置上限有10A，防止电机启动时电流过大而引起报错。

4. **4438关节的等效到电机端的转动惯量是多少**

答：电机端的转动惯量是 2.9355×10^-8 Kg.m^2

1. **电机断电单圈位置保存**

    答：电机使用的是单圈绝对值编码器，断电重新上电后，电机位置只会记录单圈位置，范围为：-0.5~+0.5圈。

    例如：转动到1.3转处，断电上电，位置记录的是0.3转，转到到0.8转处，断电上电，位置记录的是-0.2转。
