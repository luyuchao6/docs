# 4.2.3 SDK 通用问题

1. sdk 板可以接多少电机

答：1 个 can 通道，1khz 控制频率下最多控制 10 个电机，500hz 控制频率下最多控制 20 个电机，以此类推。不同 CAN 通道之间互不干扰，例如一个 4 路 CAN 叠板在 1Khz 下，每个 CAN 通道最多可以接 10 个电机，有 4 条 CAN 通道，最多一共可以接 40 个电机。

1. SDK 板在没有接开关按钮却可以给电机供电

答：开关 MOS 管坏了，没有了阻隔电源的功能了，此时注意不要再使用了，防止烧毁电机。
