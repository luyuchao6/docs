## **技术规格参数**

长140mm、宽82mm、高40mm（带外壳）

长134.8mm、宽77.3mm、21.3mm（不带壳）

1. 输入电压范围：24V-50V
2. 输出电压范围：24V-50V
3. FDCAN波特率：5Mbps
4. 7路CAN/FDCAN通道
5. 功能拓展接口都为GH_1.25_3p\4p\5p

## 主要功能以及参数介绍

### 底板

- 底板主要分为功率部分和通信部分。
    - **功率部分**：用于接入输入电源，并通过物理按键开关控制电源输出。
    - **通信部分**：用于与系统板进行通信，获取功率部分中7路FDCAN的信息，并将信息转化为串口信息，然后传输到系统板中。
![image](../assets/images/feishu-Iukhb7BdUotHzFxmKY2csD6qneb.png)

**接口说明：**

1. **电源输入接口**：采用XT60公头，支持24-48V电压范围。
2. **XT30(2+2)电机接口**：
    - 经MOS管与电源输入隔离，输出电压与输入电压一致，由下方开关控制。
    - 支持FDCAN通信，可与通信板协作，将FDCAN信息转换为串口信息及相应的CAN通道号。
    - 电机接口的CAN通道序号按图示蓝色数字顺序排列。
3. **系统板接口：** 把主板接在此处。
4. **XT30(2+2)电机控制按钮接口**：用于控制电机供电，短按进行开关操作。
5. **系统板供电按钮接口：** 用于控制系统板供电，长按进行开关操作。

### 系统板

- 系统搭载 RK3588 八核处理器（四核 Cortex-A76 + 四核 Cortex-A55）。
- 集成性能强大的 GPU 和内置 6T 算力的 NPU。
- 拥有大容量 eMMC 与 DDR 存储芯片，提供高速数据读写能力。
- 提供丰富的接口资源，包括 USB 2.0/3.0、HDMI、TF 卡插槽、Mini-PCIe（支持 4G/WiFi）、Type-C、调试串口、音频输入输出等，方便扩展与调试。

## 上电说明与信号灯状态

### 底板供电

- 在电源输入接口接入电源，此时功率部分的信号灯亮起，为绿灯常亮，蓝灯闪烁。
![底板功率部分信号灯状态](../assets/images/feishu-CskabNFPXozwjXxmhB9cHmhhn0c.png)<br>底板功率部分信号灯状态

### 系统板供电

- 长按系统板供电按钮，使系统板的系统电脑开机，此时按钮亮起，底板和系统板的信号灯亮起，功率部分为绿灯常亮，蓝灯闪烁，通信部分为红灯常亮，蓝灯闪烁。
![底部信号灯状态](../assets/images/feishu-Jv1pbuKWcomCQ4xLncrcU3bGnKb.png)<br>底部信号灯状态

- 系统板信号灯为红灯亮起，蓝灯闪烁。
![系统板信号灯状态](../assets/images/feishu-JbMtb4p7Yo4WUfxVANgcg26TnFg.png)<br>系统板信号灯状态

### 电机供电

给电机供电，并且短按一下电源接口处的按钮，此时电机底部信号灯亮起。

![开关按钮与电机信号灯状态](../assets/images/feishu-FveObygfooROGDxsjXocEeMQndb.png)<br>开关按钮与电机信号灯状态
