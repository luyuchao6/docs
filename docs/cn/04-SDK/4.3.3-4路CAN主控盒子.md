## 二.主要功能以及参数介绍

### 2.1 底板

- 底板主要分为功率部分和通信部分。
    - **功率部分**：用于接入输入电源，并通过物理按键开关控制电源输出。
    - **通信部分**：用于与系统板进行通信，获取功率部分中4路FDCAN的信息，并将信息转化为串口信息，然后传输到系统板中。
![image](../assets/images/feishu-Z08JbegowolM93xlJaJcJ7YXnyd.png)

**接口说明：**

1. **电源输入接口**：采用XT60公头，支持24-48V电压范围。
2. **XT30(2+2)电机接口**：
    - 经MOS管与电源输入隔离，输出电压与输入电压一致，由下方开关控制。
    - 支持FDCAN通信，可与通信板协作，将FDCAN信息转换为串口信息及相应的CAN通道号。
    - 电机接口的CAN通道序号按图示蓝色数字顺序排列。
3. 系统板接口，把主板接在此处。
4. **XT30(2+2)电机控制按钮接口**：用于控制电机供电，短按进行开关操作。
5. **系统板供电按钮接口：** 用于控制系统板供电，长按进行开关操作。

### 2.2 系统板

- 系统板搭载 RK3588 八核处理器（四核 Cortex-A76 + 四核 Cortex-A55）。
- 集成性能强大的 GPU 和内置 6T 算力的 NPU。
- 拥有大容量 eMMC 与 DDR 存储芯片。
- 提供丰富的接口资源，包括 USB 2.0/3.0、HDMI、TF 卡插槽、Mini-PCIe（支持 4G/WiFi）、Type-C、调试串口、音频输入输出等，方便扩展与调试。
![image](../assets/images/feishu-OjxAbPidWofs96xogAvccWdSnGh.png)

### 三. 上电说明与信号灯状态

#### 3.1底板供电

- 在电源输入接口接入电源，此时功率部分的信号灯亮起，为绿灯常亮，蓝灯闪烁。
![底部功率部分信号灯状态](../assets/images/feishu-HGIybFrU9oB8gzxzsjjcyXRZnqf.png)<br>底部功率部分信号灯状态

#### 3.2系统板供电

- 长按系统板供电按钮，使系统板的系统电脑开机，此时按钮亮起，底板和系统板的信号灯亮起，功率部分为绿灯常亮，蓝灯闪烁，通信部分为绿灯常亮，红灯闪烁。
![底部信号灯状态](../assets/images/feishu-P6cTbLni5o58NOxqHQTcyd4Fnxc.png)<br>底部信号灯状态

- 系统板信号灯为红灯亮起，蓝灯闪烁。
![系统板信号灯状态](../assets/images/feishu-GDk3bKs7fo0Bluxm1cMchW1pnXf.png)<br>系统板信号灯状态

#### 3.3电机供电

- 给电机供电，并且短按一下电源接口处的按钮，此时电机底部信号灯亮起。
![开关按钮与电机信号灯状态](../assets/images/feishu-ChLBbU3cXotkzAxBottcjyC1nOf.png)<br>开关按钮与电机信号灯状态
