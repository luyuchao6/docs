## Main Functions and Parameter Description

### Power Board

- The main function of the power board is to accept input power and control power output via a physical push-button switch.
![image](../assets/images/feishu-EVjlbISIKoexRixL2p9ceX7OnRd.png)

**Interface Details:**

1. **Power input interface**: Uses an XT60 male connector, supports a voltage range of 12–24V.
2. **XT30(2+2) motor interface**:
    - Isolated from the power input via a MOSFET; output voltage matches input voltage, controlled by the switch below.
    - Supports FDCAN communication; can work with the communication board to convert FDCAN messages into serial messages along with the corresponding CAN channel number.
    - Motor interface CAN channel numbers are arranged in the order of the blue numbers shown in the diagram.
3. **Communication board connection interface**: Used to connect the communication board.
4. **XT30(2+2) motor control button interface**: Used to control motor power supply; short-press to toggle on/off.

### Communication Board

- The communication board module must be used together with the power board. Its main function is to receive four channels of FDCAN messages from the power board and convert them into serial messages. The converted messages are output via a Type-C interface, providing four serial ports corresponding to the four CAN/FDCAN channels.
![image](../assets/images/feishu-Ys7BbbhGhol0XgxeqiBc2OplnNd.png)

- Communication methods: serial communication, FDCAN
- Serial communication baud rate: 4 Mbps;
- FDCAN baud rate: 5 Mbps;
- Upstream data interface: Type-C
- Interface description:
    1. USB connector, connects to the host to run programs and control motors.

**Connection Steps:**

1. Connect the power supply to the **power input interface**;
2. Connect the motors to the **XT30(2+2) motor interface**;
3. Insert the communication board into the **communication board interface**;
4. Connect the external push-button switch to the **motor control button interface**;
5. Connect to a computer via the **USB interface**.

## Power-On Instructions and Indicator Light Status

### Power Board Power Supply

- Connect the power supply to the XT60 power input port to power the power board. At this point, the green indicator light on the power board turns on and the blue indicator light flashes.
![image](../assets/images/feishu-P9FHbOjvAol17sxuGpgcJOyQnYb.png)

Power board indicator light status

![Power board indicator light side view](../assets/images/feishu-NcX6b5PgfolsIdxjgMfcu5mnnuh.png)<br>Power board indicator light side view

### Communication Board Power Supply

- Connect the USB cable between the communication board and the host to power the communication board. At this point, the green indicator light on the communication board turns on and the red indicator light flashes.
![Communication board indicator light status](../assets/images/feishu-ObdLbpLaHoVXErxmqu4c9sXTnOb.png)<br>Communication board indicator light status

### Motor Power Supply

- Short-press the motor power button once to light up the button; the blue indicator light at the base of the motor will then turn on.
![Switch button and motor indicator light status](../assets/images/feishu-Q08gbPnoroN5nvxZKQTcGkBrnUT.png)<br>Switch button and motor indicator light status
