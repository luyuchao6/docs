## II. Main Functions and Parameter Overview

### 2.1 Base Board

- The base board is mainly divided into a power section and a communication section.
    - **Power section**: Used to connect the input power supply and control power output via a physical push-button switch.
    - **Communication section**: Used to communicate with the system board, retrieve information from the 4-channel FDCAN in the power section, convert that information into serial data, and transmit it to the system board.
![image](../assets/images/feishu-Z08JbegowolM93xlJaJcJ7YXnyd.png)

**Interface Description:**

1. **Power input interface**: Uses an XT60 male connector, supports a voltage range of 24–48V.
2. **XT30(2+2) motor interface**:
    - Isolated from the power input via a MOSFET; output voltage matches input voltage, controlled by the switch below.
    - Supports FDCAN communication and can work with the communication board to convert FDCAN information into serial data along with the corresponding CAN channel number.
    - The CAN channel numbers of the motor interfaces are arranged in the order of the blue numbers shown in the diagram.
3. System board interface — attach the main board here.
4. **XT30(2+2) motor control button interface**: Used to control motor power supply; short-press to toggle on/off.
5. **System board power button interface**: Used to control system board power supply; long-press to toggle on/off.

### 2.2 System Board

- The system board is equipped with the RK3588 octa-core processor (quad-core Cortex-A76 + quad-core Cortex-A55).
- Integrates a high-performance GPU and a built-in NPU with 6 TOPS of compute power.
- Features large-capacity eMMC and DDR memory chips.
- Provides a rich set of interfaces including USB 2.0/3.0, HDMI, TF card slot, Mini-PCIe (supports 4G/WiFi), Type-C, debug serial port, audio input/output, etc., for convenient expansion and debugging.

### III. Power-On Instructions and Indicator Light Status

#### 3.1 Base Board Power Supply

- Connect the power supply to the power input interface. At this point, the indicator lights on the power section illuminate: the green light stays solid and the blue light blinks.
![Base power section indicator light status](../assets/images/feishu-HGIybFrU9oB8gzxzsjjcyXRZnqf.png)<br>Base power section indicator light status

#### 3.2 System Board Power Supply

- Long-press the system board power button to boot the system board's computer. At this point the button lights up, and the indicator lights on both the base board and system board illuminate. In the power section, the green light stays solid and the blue light blinks; in the communication section, the green light stays solid and the red light blinks.
![Base indicator light status](../assets/images/feishu-P6cTbLni5o58NOxqHQTcyd4Fnxc.png)<br>Base indicator light status

- The system board indicator lights show the red light on and the blue light blinking.
![System board indicator light status](../assets/images/feishu-GDk3bKs7fo0Bluxm1cMchW1pnXf.png)<br>System board indicator light status

#### 3.3 Motor Power Supply

- Power the motor and short-press the button at the power interface once. At this point the indicator light at the bottom of the motor illuminates.
![Switch button and motor indicator light status](../assets/images/feishu-ChLBbU3cXotkzAxBottcjyC1nOf.png)<br>Switch button and motor indicator light status
