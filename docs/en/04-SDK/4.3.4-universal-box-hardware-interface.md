#### Wiring Description

1. **Power Input Connector**: Uses an XT60 male connector, supports a voltage range of 12-24V.
2. **XT30(2+2) Power Output Connectors**:
    - Motor connectors, used for connecting motors.
    - Isolated from the power input via MOSFETs; output voltage matches input voltage, controlled by the switch below.
    - Supports FDCAN communication and can work with the communication board to convert FDCAN messages into serial messages along with the corresponding CAN channel number.
    - The CAN channel numbers of the motor connectors are arranged as shown: ②-⑧ correspond to CAN7-CAN1.
3. **USB Connector**: Used for data exchange between the computer and the communication board.

**Connection Steps:**

1. Connect the power supply to the **Power Input Connector**;
2. Connect the motors to the **XT30(2+2) Motor Connectors**;
3. Connect to the computer via the **USB Connector**.

#### Power-On Instructions

**Note:**

- Please power all devices when using the SDK program.
- Do not hot-plug devices while powered.

##### Power Board Power Supply

- Connect the power to the XT60 power input channel to supply the power board, then turn on the switch below. At this point, the green and red LEDs on the power board will light up, and the two blue LEDs will flash.
![Power board indicator light status](../assets/images/feishu-DS2GbAdQwoOLmtxbbQZc7T5TnGb.png)<br>Power board indicator light status

##### Motor Power Supply

- Briefly press the motor power button to light it up; at this point, the blue LED at the base of the motor will turn on.
![image](../assets/images/feishu-M01obod0eoZxuJxGxDscQiovnCe.png)
