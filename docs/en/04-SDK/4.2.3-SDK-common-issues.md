# 4.2.3 SDK Common Issues

1. How many motors can an SDK board connect to?

Answer: One CAN channel can control a maximum of 10 motors at 1 kHz control frequency, and a maximum of 20 motors at 500 Hz control frequency, and so on proportionally. Different CAN channels do not interfere with each other. For example, a 4-channel CAN expansion board at 1 kHz can connect up to 10 motors per CAN channel; with 4 CAN channels, it can connect up to 40 motors in total.

1. The SDK board can power the motors even without a switch button connected.

Answer: The switch MOSFET is damaged and no longer blocks the power supply. Stop using it immediately to prevent burning out the motors.

1. Serial port failed to open
![image](../assets/images/feishu-KYdBbHgyqo7FnxxVfZkcQVHAnRh.png)

Grant permissions and run again: sudo chmod -R 777 /dev/tty*
