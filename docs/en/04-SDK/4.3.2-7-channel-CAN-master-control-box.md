## **Technical Specifications**

Length 140mm, Width 82mm, Height 40mm (with housing)

Length 134.8mm, Width 77.3mm, 21.3mm (without housing)

1. Input voltage range: 24V-50V
2. Output voltage range: 24V-50V
3. FDCAN baud rate: 5Mbps
4. 7-channel CAN/FDCAN channels
5. Function expansion interfaces are all GH_1.25_3p\4p\5p

## Main Functions and Parameter Description

### Base Board

- The base board is mainly divided into a power section and a communication section.
    - **Power section**: Used to connect the input power supply and control the power output via a physical push-button switch.
    - **Communication section**: Used to communicate with the system board, retrieve information from the 7-channel FDCAN in the power section, convert that information into serial data, and transmit it to the system board.
![image](../assets/images/feishu-Iukhb7BdUotHzFxmKY2csD6qneb.png)

**Interface Description:**

1. **Power input interface**: Uses an XT60 male connector, supports 24-48V voltage range.
2. **XT30(2+2) motor interface**:
    - Isolated from the power input via a MOSFET; output voltage matches input voltage and is controlled by the switch below.
    - Supports FDCAN communication and can work with the communication board to convert FDCAN information into serial data along with the corresponding CAN channel number.
    - The CAN channel numbers of the motor interfaces are arranged in the order of the blue numbers shown in the diagram.
3. **System board interface:** Connect the main board here.
4. **XT30(2+2) motor control button interface**: Used to control motor power supply; short press to toggle on/off.
5. **System board power button interface:** Used to control system board power supply; long press to toggle on/off.

### System Board

- The system is equipped with the RK3588 octa-core processor (quad-core Cortex-A76 + quad-core Cortex-A55).
- Integrates a high-performance GPU and an NPU with 6 TOPS of built-in computing power.
- Features large-capacity eMMC and DDR memory chips, providing high-speed data read/write capability.
- Offers rich interface resources including USB 2.0/3.0, HDMI, TF card slot, Mini-PCIe (supports 4G/WiFi), Type-C, debug serial port, audio input/output, etc., for convenient expansion and debugging.

## Power-On Instructions and Indicator Light Status

### Base Board Power Supply

- Connect the power supply to the power input interface. At this point, the indicator lights on the power section of the base board will turn on: the green light stays solid and the blue light blinks.
![Base board power section indicator light status](../assets/images/feishu-CskabNFPXozwjXxmhB9cHmhhn0c.png)<br>Base board power section indicator light status

### System Board Power Supply

- Long press the system board power button to boot the system computer on the system board. At this point, the button lights up, and the indicator lights on both the base board and system board turn on. The power section shows a solid green light and blinking blue light; the communication section shows a solid red light and blinking blue light.
![Bottom indicator light status](../assets/images/feishu-Jv1pbuKWcomCQ4xLncrcU3bGnKb.png)<br>Bottom indicator light status

- The system board indicator light shows a solid red light and a blinking blue light.
![System board indicator light status](../assets/images/feishu-JbMtb4p7Yo4WUfxVANgcg26TnFg.png)<br>System board indicator light status

### Motor Power Supply

Power the motor and short press the button at the power interface once; the indicator light at the bottom of the motor will turn on.

![Switch button and motor indicator light status](../assets/images/feishu-FveObygfooROGDxsjXocEeMQndb.png)<br>Switch button and motor indicator light status
